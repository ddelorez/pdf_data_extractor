Overview

This is a Flask-based REST API for extracting production data from PDF documents. The system processes PDF files, extracts well production records, validates the data, and outputs results in Excel and CSV formats.

  • PDF file upload with batch processing support
  • Data extraction using Phase 1 extraction engine
  • Automatic data validation and deduplication
  • Excel and CSV output generation
  • Job tracking with real-time status updates
  • Docker containerization for scalable deployment

API Endpoints

POST /api/extract

Upload PDF files for extraction

curl -F "files=@document.pdf" http://localhost:5000/api/extract
POST /api/process/<job_id>

Process uploaded PDFs and generate outputs

curl -X POST http://localhost:5000/api/process/job-uuid
GET /api/status/<job_id>

Check processing status and progress

curl http://localhost:5000/api/status/job-uuid
GET /download/<job_id>/output.xlsx

Download processed Excel file

GET /download/<job_id>/output.csv

Download processed CSV file

GET /api/health

Health check for Docker/Kubernetes monitoring

curl http://localhost:5000/api/health

Quick Start

  1. Upload files: Use POST /api/extract with multipart form data
  2. Process: Call POST /api/process/{job_id} with the returned job_id
  3. Check status: Monitor progress with GET /api/status/{job_id}
  4. Download: Get results from /download/{job_id}/output.xlsx or .csv
